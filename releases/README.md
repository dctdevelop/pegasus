# Pegasus Release Notes - Summary

## Versioning
Name reflects ```major```, ```minor``` and ```patch``` concepts:
```
	major.minor.patch
```

Breaking changes may occur on ```major``` and ```minor``` numbers. 
```patches```, are not expected to affect compatibility.
